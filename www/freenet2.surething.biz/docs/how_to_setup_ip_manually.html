FreeNet manual network switching instructions

These instructions are for a select group of people that want the ability to select the internet provider they want to use over the FreeNet network system. The FreeNet network at the time this was written has two completely separate paths to get onto the network. Normally when you hook to FreeNet our system provides you with a path to one of these two paths using DHCP FreeNet provides your computer with the needed info to make the connection to one of them eather TOT or True automaticaly. FreeNet also auto detects if one of these networks fails we will switch all the users onto the remaining working Internet path. However there are times that some people want to switch to eather The TOT network path or the True network path manually, in this case this is the information needed to manually setup your computer to use one or the other. Before you do any manual IP setup on FreeNet you might want to take a look at the FreeNet network usage activity to select what network is being least used at the times you plan to use the network. This can bee seen in a graphical format of bytes/secound verses time at the local FreeNet server http://surething.biz/cacti/graph_view.php?action=tree&tree_id=1 Both TOT and True usage have graphs there for you to see.


How to find your FreeNet provided IP address

Before you change your gateway address you need to find out what IP address FreeNet is giving you in the auto IP mode first. This can be done by going to the command prompt in Vista or Windows XP and entering the command “ipconfig” in the command window. To bring up a command prompt in Windows Vista, From the Start Menu, type Command Prompt in the search field then click Command Prompt once it appears. In the command window type “ipconfig” no quotes and look for the ipv4 address in one of the many lines of info provided from the command entered. Record this four digit number 192.168.2.XXX address to be used in the manual settings bellow.


To manualy switch to the TOT or True ISP provider

There are four numbers you need to change or setup in your computer to manually setup your IP and gateway manually. Note only the gateway address number changes to move from TOT to True.


TOT numbers

1. your IP address:192.168.2.XXX

you should find out what your IP address is set to in auto mode before you change to manual mode. Your IP address must be the same as auto or you will not have Internet access only local access as noted in How to find your FreeNet provided IP address above.

2. mask address: 255.255.255.0

3. gateway address: 192.168.2.253

4. DNS server address: 192.168.2.250


True numbers

1. your IP address:192.168.2.XXX

you should find out what your IP address is set to in auto mode before you change to manual mode. Your IP address must be the same as auto or you will not have Internet access only local access.

2. mask address: 255.255.255.0

3. gateway address: 192.168.2.248

4. DNS server address: 192.168.2.250



To change IP address and gateway in Microsoft Vista


Step 1

Go to the Control Panel from the Start menu.

Step 2

Click "Network and Internet" and then "Network and Sharing Center."

Step 3

Click "Manage network connections."

Step 4

Choose which Internet connection you would like to change if you have more than one. Right click on it and select "Properties."

Step 5

Click the "Networking" tab.

Step 6

Choose Internet Protocol Version 4 (TCP/IPv4) .

Step 7

Click "Use the following IP Address" to enter your own IP settings if you want to have manual control of your address.


Step 8

If you want to reverse and go back to auto mode skip step 7 and do this step otherwise skip this step and go to step 9.

Click "Obtain an IP Address Automatically" to reset your IP address automatically, then click OK.


Step 9

Enter your IP address as you noted it was set when in auto mode. into the boxes marked "IP Address," for "Subnet Mask" use 255.255.255.0 and "Default gateway." use 192.168.2.253



Change your DNS Server Settings

Step 1

For automatic setting or reversing manual settings

Click "Obtain DNS Server Address Automatically" and then click "OK."

skip this and go to step 2 if going to manual settings

Step 2

For manual settings start here.

Click "Use the following DNS Server addresses" to change your DNS Server settings manually.

Step 3

Type your primary DNS Server address into the box that says "Preferred DNS Server."

enter 192.168.2.250 for the FreeNet DNS server or you can use opendns server numbers to bypass the FreeNet DNS server with 208.67.222.222 or others if you know of any others.

Step 4

Type your secondary DNS Server address into the box that says "Alternate DNS Server."

here again you might want to use the opendns server as a backup instead of the primary 208.67.222.222 or can just leave blank or swap this with the 192.168.2.250 FreeNet address
